⚡ Deterministic Finality

No fork choice. No live longest-chain rule. The node processes only height = finalized_height + 1. Every block is final.

🔄 Adaptive Committee

Validator committee size, min bond, and eligibility thresholds adjust deterministically based on qualified operator depth.

📦 UTXO Ledger

Simple, auditable UTXO model with P2PKH addresses. Ed25519 signatures. Deterministic reward settlement.

📡 Light Client Support

JSON-RPC lightserver exposes finalized-state queries. REST explorer and desktop wallet included.

🚀 Quick Start

# Clone and build
git clone https://github.com/finalis-core/finalis-core
cd finalis-core
cmake -S . -B build -G Ninja
cmake --build build -j

# Run a node with lightserver
./build/finalis-node --db ~/.finalis/mainnet --with-lightserver

Default ports: P2P 19440 · Lightserver 19444
