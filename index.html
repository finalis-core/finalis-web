🚀 Quick Start (5 minutes)

# Clone and build
git clone https://github.com/finalis-core/finalis-core
cd finalis-core
cmake -S . -B build -G Ninja && cmake --build build -j

# Run your mainnet node
./build/finalis-node --db ./node_data --network mainnet
